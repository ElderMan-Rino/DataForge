# MyWinFormsApp

## 프로젝트 개요  
이 프로젝트는 다양한 데이터 형식을 읽고 변환한 후, 데이터를 저장하는 WinForms 애플리케이션입니다. 사용자는 엑셀 파일 등에서 데이터를 읽고, 이를 특정 형식으로 변환하여 저장할 수 있습니다.

---

## 폴더 구조  

### **/Core**  샘플
`Core` 폴더는 애플리케이션의 핵심 로직과 공용 리소스를 담고 있는 영역으로, 프로젝트 전반에 걸쳐 재사용 가능한 구성 요소가 위치합니다.  
- **Constants.cs**: 프로젝트에서 사용되는 상수 값을 정의합니다.  
- **Enums.cs**: 데이터 처리 및 변환에서 사용하는 열거형(Enum)을 정의합니다.  
- **Interfaces**: 서비스 및 매니저에서 구현해야 할 인터페이스 정의를 포함합니다.  
  - **IDataLoader.cs**: 데이터 로딩 기능을 정의하는 인터페이스입니다.  
  - **IDataSaver.cs**: 데이터 저장 기능을 정의하는 인터페이스입니다.  

### **/View**  샘플
`View` 폴더는 사용자 인터페이스와 관련된 모든 WinForms 폼 및 컨트롤이 위치하는 영역입니다.  
- **MainForm.cs**: 애플리케이션의 메인 윈도우 폼입니다. 데이터 로딩, 변환, 저장 등의 UI 작업을 제공합니다.  
- **Dialogs**: 사용자와의 상호작용을 위한 다이얼로그 폼이 위치합니다.  
  - **FileSelectionDialog.cs**: 파일 선택 다이얼로그를 구현한 폼입니다.  
  - **SettingsDialog.cs**: 애플리케이션 설정과 관련된 다이얼로그입니다.  
- **Controls**: 재사용 가능한 WinForms 사용자 컨트롤이 위치합니다.  
  - **ProgressBarControl.cs**: 진행률 표시를 위한 커스텀 컨트롤입니다.  

### **/Services**  
이 폴더는 데이터 로딩, 변환, 저장 등 주요 작업을 담당하는 서비스 클래스들이 위치합니다.  
- **DataLoaderService.cs**: 데이터를 외부 파일(예: 엑셀, CSV 등)에서 읽어오는 기능을 제공합니다.  
- **DataSaverService.cs**: 데이터를 특정 형식으로 저장하는 기능을 담당합니다.  
- **ConversionService.cs**: 데이터를 다른 형식으로 변환하는 로직을 구현합니다.  

### **/Managers**  
이 폴더에는 여러 서비스를 조정하거나, 여러 단계를 거치는 비즈니스 로직을 관리하는 클래스들이 위치합니다.  
- **DataProcessingManager.cs**: 데이터 처리 전반을 관리하는 매니저 클래스입니다. 여러 서비스(예: 데이터 로딩, 변환, 저장)를 조정합니다.  

### **/Logic**  샘플
이 폴더는 비즈니스 로직을 담당하는 클래스들이 위치합니다. 데이터 처리, 계산, 변환 등 복잡한 로직을 이 폴더에 정의합니다.  
- **DataConversionLogic.cs**: 데이터를 변환하는 로직을 담당하는 클래스입니다.  
- **DataProcessingLogic.cs**: 데이터 처리 및 비즈니스 로직을 관리하는 클래스입니다.  

### **/Helpers**  샘플
이 폴더는 반복적인 작업이나 단순한 로직을 처리하는 헬퍼 클래스들이 포함됩니다.  
- **FileHelper.cs**: 파일 입출력과 관련된 작업을 돕는 클래스입니다.  
- **ValidationHelper.cs**: 데이터 유효성 검사와 관련된 헬퍼 클래스입니다.  

### **/Data**  샘플
이 폴더는 데이터를 저장하고 전달하는 데이터 객체들이 위치합니다.  
- **RawData.cs**: 읽어온 원시 데이터를 담는 클래스입니다.  
- **ConvertedData.cs**: 변환된 데이터를 담고 있는 클래스입니다.  

---

## 설치 및 사용법  
1. **Visual Studio**에서 프로젝트를 클론합니다.  
2. 필요한 라이브러리를 설치한 후, 애플리케이션을 실행할 수 있습니다.  

---

## 주요 기능  
- 엑셀 데이터를 읽어오는 기능  
- 데이터를 변환하는 기능  
- 변환된 데이터를 저장하는 기능
